속도를 적분하면 거리..
가속도를 적분하면 속도..

위치 = 초기 위치 + 초기 속도 * 시간
속도 = 초기 속도 + 가속도 * 시간

위치 = 초기 위치 + 초기 속도 * 시간 + ½ * 가속도 * 시간²

오늘의 목표 : 가속도 추가 -> 파티클의 생성되는 시작 시간, 유지되는 시간 추가 -> 반복
